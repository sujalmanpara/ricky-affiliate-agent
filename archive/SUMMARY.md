## 🔧 Tech Stack

### APIs Used

**1. Amazon Product Advertising API**
- **Purpose:** Product research & affiliate links
- **Cost:** Free for affiliates
- **Limits:** 1 req/sec, 8,640 req/day (more than enough)

**2. Postiz API**
- **Purpose:** Multi-platform social media posting
- **Cost:** $19-39/month (unlimited posts)
- **Platforms:** Instagram, Twitter, Pinterest, Facebook, LinkedIn

**3. Image Generation AI (Choose One)**

**Option A: Fal.ai (Recommended)**
- **Cost:** $0.01-0.03 per image
- **Speed:** 2-4 seconds
- **Models:** FLUX Pro, FLUX Dev, Stable Diffusion
- **Best for:** Fast, high-quality enhancement

**Option B: Replicate**
- **Cost:** $0.02-0.05 per image
- **Speed:** 5-10 seconds
- **Models:** SDXL, Kandinsky, FLUX
- **Best for:** Flexibility, multiple models

## 🎨 Image Generation Flow

### Scenario 1: Product Image Available (80% of cases)
```
1. Download product image from Amazon
2. Call Fal.ai image-to-image API
3. Enhance with prompt:
   "Professional product photography, {product}, 
   minimalist background, soft lighting, 4k quality"
4. Add text overlays (price, CTA)
5. Save 5 style variations
```

### Scenario 2: No Product Image (20% of cases)
```
1. Extract product title + description
2. Call Fal.ai text-to-image API
3. Generate with prompt:
   "High-quality product photo of {product}, 
   {description}, professional lighting, 4k"
4. Add overlays
5. Save variations
```
